For cloth Try-On

Download and install Lens Studio

Login using your Snapchat Account

Click on open project

Select Virtual Try On

In the top right corner, use send to snapchat to use the try-on on your phone.


For shoe Try-On

open shoetryon-web-js-main folder
Open terminal in this folder and run the following commands

- Run `npm install`
- Run `npm run demo`


Note:- Install node.js before running these commands.
